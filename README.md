# Demoparse

GoldSrc and Source demo parser based on [compLexity Demo Player](https://code.google.com/p/coldemoplayer/).

## Installation

Add this line to your application's Gemfile:

    gem 'demoparse'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install demoparse
